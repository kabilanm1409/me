# Kabilan M Portfolio

A clean, modern, recruiter-friendly portfolio built with HTML, CSS, and JavaScript.

## Design Style

- Light, minimal layout with optional dark mode
- Blue, teal, and indigo accent palette
- Modern gradient hero section
- Smooth scroll and subtle animations
- Responsive layout for mobile and desktop
- Fast-loading static structure

## Sections

- Home with photo, introduction, and resume download
- About with professional profile, education, skills, experience, and achievements
- Projects with clean cards and filtering
- Contact with links and a contact form

## Files

- `index.html` - landing page
- `style.css` - visual design and responsive layout
- `script.js` - typing text, theme toggle, scroll progress, filters, and form behavior
- `pages/about.html` - profile and background
- `pages/projects.html` - featured projects
- `pages/contact.html` - contact information and form
- `assets/` - resume and project/profile media

## Notes

Replace the placeholder name, email, phone number, and social links with your real details before publishing. Add your actual `profile.jpg`, `resume.pdf`, `project1.png`, and `project2.png` files into `assets/` when they are ready.
